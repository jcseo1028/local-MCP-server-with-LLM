# VSIX v1.1.1 — 런타임 버그 수정

## 날짜
2026-04-09

## 변경 사유
사용자 테스트에서 두 가지 런타임 버그 발견:
1. Markdown 렌더링 실패 — Markdig.dll 어셈블리 로딩 불가
2. ComboBox 도구명이 Dark Theme에서 거의 보이지 않음

## 변경 내역

### 1. `LocalMcpVsExtensionPackage.cs`
- `[ProvideBindingPath]` 어트리뷰트 추가 → VS가 확장 설치 디렉터리를 어셈블리 프로빙 경로에 포함하여 Markdig.dll 로딩 가능
- `InstalledProductRegistration` 버전을 `1.1.0`으로 명시

### 2. `SummaryToolWindowControl.cs`
- ComboBox 배경색: `VsBrushes.ComboBoxBackgroundKey` → `VsBrushes.WindowKey` (Dark Theme 호환)
- ComboBoxItem 생성 시 `SetResourceReference(ForegroundProperty, VsBrushes.WindowTextKey)` 추가
- `ShowMarkdown()`: silent `catch` → `catch (Exception ex)` + 오류 메시지 표시로 변경 (디버깅 가능)

## 영향 범위
- `src/LocalMcpVsExtension/` — Package, ToolWindow 2개 파일
- `.agents/changes/` — 본 기록 추가
