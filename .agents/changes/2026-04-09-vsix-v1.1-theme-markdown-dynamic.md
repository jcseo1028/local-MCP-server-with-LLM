- VS 2022 Dark/Light/Blue 테마를 자동 감지하여 Tool Window UI에 반영
- LLM 응답(Markdown)을 FlowDocument로 파싱/렌더링 (Markdig 라이브러리)

## 기술 결정 근거

### Markdown 렌더링: Markdig + FlowDocument (WebView2 대신)
- Markdig: 순수 .NET 라이브러리, 네이티브 DLL 불필요 → VSIX 패키징 안정
- FlowDocument: WPF 기본 컨트롤, VS 테마 색상과 자연스러운 통합
- FlowDocumentScrollViewer: 내장 텍스트 검색(Ctrl+F), 복사(Ctrl+C) 지원
- WebView2: 네이티브 DLL 포함 시 VSIX 패키징 복잡화 → 불채택

### VS 테마 적용 방식
- UI 컨트롤: `VsBrushes.*Key` SetResourceReference → 런타임 테마 변경 자동 반영
- FlowDocument: `VSColorTheme.GetThemedColor()` → 실제 Color 값으로 스타일 생성
- 코드블록/인라인코드: 테마 밝기 감지 후 적절한 배경색 자동 계산

### 동적 도구 로딩
- `Loaded` 이벤트에서 `GET /api/tools/list` 호출
- 서버 미접속 시 `summarize_current_code` 폴백 등록
- ComboBox 선택 기반으로 `CallToolAsync(toolName, arguments)` 호출
