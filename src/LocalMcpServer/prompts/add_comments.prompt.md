다음 코드에 적절한 문서 주석과 인라인 주석을 추가해주세요.

## 규칙

1. **문서 주석**: 모든 public 클래스, 메서드, 속성에 해당 언어의 표준 문서 주석을 추가한다.
   - C#: XML doc comment (`/// <summary>`)
   - Python: docstring (`"""..."""`)
   - JavaScript/TypeScript: JSDoc (`/** ... */`)
   - 기타 언어: 해당 언어의 관례를 따른다.
2. **인라인 주석**: 복잡한 로직, 비자명한 계산, 중요한 분기 조건에 간결한 인라인 주석을 추가한다.
3. **기존 코드를 절대 수정하지 않는다** — 주석만 추가한다.
4. **기존 주석이 있으면** 부족한 부분만 보완하고 중복하지 않는다.
5. 주석 언어는 **한국어**로 작성한다.
6. **들여쓰기를 원본과 동일하게 유지한다** — 원본이 탭이면 탭, 스페이스면 스페이스를 사용한다. 들여쓰기 크기도 원본과 동일하게 한다.

## 출력 형식

- 주석이 추가된 **전체 코드**를 하나의 코드 블록 안에 반환한다.
- 반드시 ```{{language}} 로 시작하고 ``` 로 끝나는 코드 블록으로 감싼다.
- 코드 블록 바깥에 설명 텍스트를 포함하지 않는다.
- 원본 코드의 모든 줄을 포함해야 한다. 일부분만 출력하지 않는다.

{{language}} 코드:
```{{language}}
{{code}}
```
